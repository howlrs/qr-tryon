# 使用書（ユーザーガイド）

本システムの操作方法を、管理者（店舗スタッフ）と来店客それぞれの視点で説明する。

---

## 1. 管理者向けガイド

管理画面には `/{locale}/admin` からアクセスする。左側のサイドバーで各機能に遷移できる。

### 1.1 商品一覧の確認

管理画面トップ（`/admin`）にアクセスすると、登録済み商品の一覧がテーブル形式で表示される。

表示項目:
- **商品名** — 商品の名称
- **バリエーション数** — 登録済みバリエーションの件数
- **価格帯** — 最低価格 〜 最高価格（バリエーション間で異なる場合は範囲表示）
- **在庫** — 全バリエーションの合計在庫数（在庫あり: 緑、在庫なし: 赤）
- **操作** — QRコード表示・編集・削除ボタン

### 1.2 商品の新規登録

1. 商品一覧画面の右上「商品を追加」ボタンをクリック
2. 商品登録フォーム（`/admin/products/new`）が開く
3. 以下の情報を入力する

**基本情報:**
| 項目 | 必須 | 説明 |
|------|------|------|
| 商品名 | はい | 商品の表示名 |
| 商品説明 | はい | 商品の詳細説明文 |

**バリエーション（1つ以上必須）:**
| 項目 | 必須 | 説明 |
|------|------|------|
| バリエーション名 | はい | 例: "Size M / White" |
| SKU | はい | 在庫管理用のユニークコード |
| 価格 | はい | 日本円（整数） |
| 在庫数 | はい | 現在の在庫数 |
| サイズ | いいえ | サイズ表記 |
| 色 | いいえ | 色名 |

- 「バリエーションを追加」で行を追加できる
- ゴミ箱アイコンで行を削除できる（最低1行は必要）

**画像（1つ以上必須）:**
| 項目 | 必須 | 説明 |
|------|------|------|
| 画像URL | はい | 画像の外部URL（https://...） |
| 代替テキスト | いいえ | アクセシビリティ用の画像説明 |
| 表示順 | いいえ | 数値が小さいほど先に表示（デフォルト: 1） |

4. 「作成」ボタンをクリックして保存

### 1.3 商品の編集

1. 商品一覧の操作列にある鉛筆アイコンをクリック
2. 編集フォーム（`/admin/products/{id}/edit`）が開く
3. 既存データが事前入力された状態でフォームが表示される
4. 必要な項目を変更し「更新」ボタンをクリック

### 1.4 商品の削除

1. 商品一覧の操作列にあるゴミ箱アイコンをクリック
2. 確認ダイアログが表示される
3. 「OK」をクリックすると商品が削除される

> **注意:** 削除は取り消せない。慎重に操作すること。

### 1.5 QRコードの生成と印刷

1. 商品一覧の操作列にあるQRコードアイコンをクリック
2. モーダルが開き、商品専用のQRコードが表示される
3. QRコードの下に対応するURLが表示される
4. 「Download QR Code」ボタンでPNG画像としてダウンロード
5. ダウンロードした画像を印刷し、店舗の商品棚やタグに貼付する

QRコードは商品詳細ページ（`/products/{id}`）へのリンクを埋め込んでおり、エラー訂正レベル「H」（高）で生成される。多少の汚れや破損があっても読み取り可能。

### 1.6 注文（リクエスト）管理

サイドバーの「注文管理」をクリックして注文一覧（`/admin/orders`）を開く。

**表示項目:**
- **ID** — リクエストの一意識別子
- **商品名** — リクエスト対象の商品
- **バリエーション名** — リクエスト対象のバリエーション
- **Type** — 「試着」または「購入」
- **時間** — リクエスト送信日時

**未読通知:**
- サイドバーの「注文管理」の横に未読リクエスト数がバッジで表示される
- バッジは2秒間隔で自動更新される
- 「既読にする」ボタンで全リクエストを既読にできる

### 1.7 言語切り替え

URLのロケールセグメントを変更することで言語を切り替えられる。

- 日本語: `/ja/admin`
- 英語: `/en/admin`

デフォルトは日本語（`ja`）。

## 2. 来店客向けガイド

### 2.1 QRコードから商品を確認する

1. 店舗に掲示されたQRコードをスマートフォンのカメラで読み取る
2. 商品詳細ページが自動的に開く
3. 商品画像・説明・バリエーション情報を確認できる

### 2.2 商品画像の閲覧

- メイン画像が大きく表示される
- 画像下のサムネイル一覧から別の画像を選択できる
- サムネイルをタップするとメイン画像が切り替わる

### 2.3 バリエーションの選択

- 「バリエーションを選択」セクションで希望のサイズ・色を選ぶ
- 選択中のバリエーションは枠線がハイライトされる
- 各バリエーションの在庫状況が表示される（「在庫: N」または「在庫切れ」）

### 2.4 試着リクエストの送信

1. 希望のバリエーションを選択
2. 「試着する」ボタンをタップ
3. 確認メッセージが表示される
4. 店舗スタッフの管理画面にリクエストが通知される

### 2.5 購入リクエストの送信

1. 希望のバリエーションを選択
2. 「購入する」ボタンをタップ
3. 確認メッセージが表示される
4. 店舗スタッフの管理画面にリクエストが通知される

### 2.6 連続リクエストの制限

同一端末から同一商品・同一バリエーション・同一タイプ（試着/購入）のリクエストは、**1分間に1回**に制限されている。制限中にリクエストを送ると「リクエスト済みです」のメッセージが表示される。

この制限は端末ごとに付与されるユニークID（`localStorage` に保存）で管理される。

### 2.7 商品一覧の閲覧

QRコードを使わずに商品一覧ページ（`/products`）にアクセスすることもできる。商品カードをタップすると商品詳細ページに遷移する。

## 3. トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| QRコードが読み取れない | QRコードの印刷品質が低い | 高解像度で再印刷する |
| 試着/購入ボタンが押せない | バリエーション未選択または在庫切れ | バリエーションを選択し、在庫を確認する |
| 「リクエスト済みです」と表示される | 1分以内の連続リクエスト | 1分待ってから再度リクエストする |
| 管理画面に通知が来ない | API通信エラー | ブラウザのコンソールでエラーを確認する |
| 画像が表示されない | 画像URLが無効 | 商品編集で正しいURLを設定する |
